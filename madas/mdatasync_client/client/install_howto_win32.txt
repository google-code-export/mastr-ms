Copy client and supportwin32 over to the windows machine.
Install Python26
Install NSIS
Install 7Zip
Extract the EnvVarUpdate extension
Copy the extension into c:\program files\nsis\include
Open a shell
Change dir to the 'client' dir
run: c:\python26\python.exe setup.py py2exe
This creates your payload in dist
Now from windows explorer, right click on the installer.nsi file and choose 'Compile with NSIS'
This creates your installer.
